TUTORIAL PENGGUNAAN

Operasi Dasar ▶
  • Tombol Angka (0-9): Tekan untuk memasukkan angka. Dapat diinput berurutan untuk membentuk bilangan.
  • Titik Desimal (.): Untuk memasukkan bilangan desimal. Contoh: 3.14
  • Operator + (Plus): Penjumlahan. Contoh: 5+3=8
  • Operator − (Minus): Pengurangan. Contoh: 9−4=5
  • Operator × (Kali): Perkalian. Contoh: 6×7=42
  • Operator ÷ (Bagi): Pembagian. Contoh: 15÷3=5
  • Tombol = (Sama Dengan): Menghitung dan menampilkan hasil operasi yang telah diinput.
  • Tombol AC (All Clear): Menghapus semua input dan hasil, mereset kalkulator ke keadaan awal.
  • SHIFT + AC (Clear Entry): Menghapus input saat ini tanpa mengubah nilai memory.
  • Tombol DEL (Delete): Menghapus satu karakter terakhir dari input.
Contoh Penggunaan:
125 + 75 = → Hasil: 200
Kurung dan Prioritas Operasi ▶
  • Kurung Buka ( : Memulai ekspresi yang akan dihitung lebih dulu.
  • Kurung Tutup ) : Menutup ekspresi dalam kurung.
  • Prioritas Operasi: Kalkulator mengikuti aturan matematika standar:
    1. Kurung ( ) → dihitung paling awal
    2. Pangkat (^) dan Faktorial (!)
    3. Perkalian (×) dan Pembagian (÷)
    4. Penjumlahan (+) dan Pengurangan (−)
Contoh Tanpa Kurung:
2 + 3 × 4 = → Hasil: 14 (bukan 20)
Contoh Dengan Kurung:
(2 + 3) × 4 = → Hasil: 20
Mode Display dan Setup ▶
  • Tombol MODE: Mengubah mode tampilan hasil perhitungan. Tekan berulang untuk beralih antar mode:
    • NORM - Normal (angka biasa)
    • FIX - Fixed decimal (jumlah angka desimal tetap)
    • SCI - Scientific notation (notasi ilmiah)
  • SHIFT + MODE (SETUP): Membuka menu pengaturan untuk mengkonfigurasi:
    • Jumlah digit desimal untuk mode FIX
    • Jumlah digit signifikan untuk mode SCI
    • Mode sudut (DEG/RAD/GRAD)
    • NORM (kembali ke tampilan normal)
  • Indikator Angle Mode: Menunjukkan mode sudut aktif (DEG/RAD/GRAD) di layar atas.
Catatan: Mode DEG (Derajat) paling umum digunakan. RAD (Radian) untuk perhitungan kalkulus, GRAD (Gradian) untuk survei geodesi.
Sistem DMS (Derajat-Menit-Detik) ▶

Format DMS: Sistem koordinat sudut yang digunakan dalam astronomi, navigasi, dan survei.

  • Tombol °'″: Untuk memasukkan nilai dalam format DMS. Tekan tombol ini untuk memasukkan simbol derajat (°), menit ('), dan detik (″).
  • Format Input DMS:
    • Derajat: 45°
    • Derajat-Menit: 45°30'
    • Derajat-Menit-Detik: 45°30'15"
  • SHIFT + °'″ (DMS→): Mengkonversi nilai DMS ke format desimal.
  • Nilai Negatif DMS: Gunakan tombol (-) sebelum nilai DMS. Contoh: (-)0°19'40"
Contoh Operasi DMS:
12°30' + 5°45'20" → Hasil: 18°15'20"
Contoh Konversi:
45°30'0" → Tekan SHIFT+°'″ → Hasil: 45.5
Penting: Pada kalkulator ini, tombol DMS bekerja secara berurutan, yaitu dimulai dari derajat, kemudian menit, dan dilanjutkan detik. Namun dalam kondisi tertentu ketika input hanya terdiri dari dua komponen (misalnya derajat dan menit) lalu langsung diikuti tanda operasi seperti tambah, kurang, kali, atau bagi, maka pada penggunaan DMS berikutnya sistem akan langsung membaca sebagai detik karena urutannya berlanjut. Untuk mengatasinya, pengguna harus menghapus input terlebih dahulu, kemudian menekan kembali agar pembacaan dimulai lagi dari derajat.
Fungsi Trigonometri ▶
  • sin (Sinus): Menghitung nilai sinus dari sudut.
    Cara: Tekan sin lalu masukkan sudut dalam kurung
    Contoh: sin(30) → Hasil: 0.5 (dalam mode DEG)
  • cos (Cosinus): Menghitung nilai cosinus dari sudut.
    Contoh: cos(60) → Hasil: 0.5 (dalam mode DEG)
  • tan (Tangen): Menghitung nilai tangen dari sudut.
    Contoh: tan(45) → Hasil: 1 (dalam mode DEG)
  • SHIFT + sin (sin⁻¹): Arc sinus / invers sinus. Menghitung sudut dari nilai sinus.
    Contoh: sin⁻¹(0.5) → Hasil: 30°
  • SHIFT + cos (cos⁻¹): Arc cosinus / invers cosinus.
    Contoh: cos⁻¹(0.5) → Hasil: 60°
  • SHIFT + tan (tan⁻¹): Arc tangen / invers tangen.
    Contoh: tan⁻¹(1) → Hasil: 45°
Perhatian: Hasil fungsi trigonometri bergantung pada mode sudut aktif (DEG/RAD/GRAD). Pastikan mode sudut sudah sesuai sebelum menghitung.
Fungsi Matematika Lanjutan ▶
  • √ (Akar Kuadrat): Menghitung akar kuadrat.
    Contoh: sqrt(25) → Hasil: 5
  • SHIFT + √ (³√): Akar pangkat tiga (kubik).
    Contoh: cbrt(27) → Hasil: 3
  • ^ (Pangkat): Menghitung perpangkatan.
    Contoh: 2^8 → Hasil: 256
    Contoh: 5^3 → Hasil: 125
  • x! (Faktorial): Menghitung faktorial (n! = n × (n-1) × ... × 1).
    Contoh: 5! → Hasil: 120
    Catatan: Hanya untuk bilangan bulat positif
  • log (Logaritma Basis 10): Menghitung logaritma basis 10.
    Contoh: log(100) → Hasil: 2
  • SHIFT + log (10ˣ): Menghitung 10 pangkat x.
    Contoh: pow10(2) → Hasil: 100
  • ln (Logaritma Natural): Menghitung logaritma basis e.
    Contoh: ln(2.718281828) → Hasil: ≈1
  • SHIFT + ln (eˣ): Menghitung e pangkat x.
    Contoh: exp(1) → Hasil: ≈2.718
Konstanta Matematika ▶
  • π (Pi): Konstanta Pi ≈ 3.14159265359
    Gunakan untuk perhitungan lingkaran, trigonometri
    Contoh: 2×π×10 → Keliling lingkaran radius 10
  • SHIFT + π (e): Konstanta Euler ≈ 2.71828182846
    Basis logaritma natural, digunakan dalam kalkulus
    Contoh: e^2 → Hasil: ≈7.389
Tips: Konstanta ini sudah tersimpan dengan presisi tinggi dalam kalkulator. Tidak perlu memasukkan nilai manual.
Memory dan Penyimpanan ▶
  • STO (Store): Menyimpan hasil perhitungan terakhir ke dalam memory.
    Cara: Hitung → tekan = → tekan STO
    Indikator memory akan menunjukkan nilai tersimpan
  • SHIFT + STO (RCL / Recall): Memanggil kembali nilai dari memory untuk digunakan dalam perhitungan.
    Cara: Tekan SHIFT → STO untuk memasukkan nilai memory ke layar
  • Ans (Answer): Menggunakan hasil perhitungan terakhir dalam operasi baru.
    Contoh: 5+3=8 → lalu Ans×2=16
  • SHIFT + Ans (PREV): Menampilkan kembali ekspresi perhitungan sebelumnya.
  • VAR (Variable): Recall variabel tersimpan (A, B, C, D, E, F, x, y, M) ke ekspresi.
  • SHIFT + VAR: Menyimpan nilai saat ini ke salah satu variabel (A, B, C, D, E, F, x, y, M).
Contoh Penggunaan Memory:
1. Hitung: 25×4=100
2. Tekan STO (tersimpan)
3. Hitung: 50÷2=25
4. Tekan SHIFT+STO untuk recall → 100
5. Operasi: +25=125
Tutorial Tombol VAR:
1. Hitung nilai yang ingin disimpan, contoh: 12+8=20
2. Tekan SHIFT lalu VAR
3. Pilih nama variabel (misalnya A) pada prompt → tersimpan sebagai A=20
4. Untuk memanggil kembali, tekan VAR lalu pilih A
5. Nilai variabel masuk ke layar dan bisa dipakai operasi, contoh: A×3=60
Cara Membuat Variabel:
Variabel dibuat saat Anda menyimpan nilai ke salah satu nama variabel (A, B, C, D, E, F, x, y, M) melalui SHIFT+VAR.
Jika variabel belum pernah disimpan, saat dipanggil dengan VAR sistem akan menampilkan pesan bahwa variabel belum ada.
Fitur dan Tombol Khusus ▶
  • (-) (Negasi): Membuat bilangan negatif. Berbeda dengan operator minus (−).
    Contoh: (-)5 → -5
    Untuk operasi: 10+(-)5=5 (sepuluh plus negatif lima)
  • SHIFT (Yellow): Mengaktifkan fungsi sekunder (warna kuning di atas tombol). Indikator SHIFT akan menyala di layar.
  • ALPHA (Red): Mengaktifkan fungsi alfa untuk memasukkan variabel. Indikator ALPHA akan menyala di layar.
  • ENG→ (Engineering): Mengubah tampilan ke notasi engineering (kelipatan 10³).
    Contoh: 1500 → 1.5×10³
  • SHIFT + ENG (←ENG): Kembali dari notasi engineering ke normal.
  • a b/c (Fraction): Input pecahan campuran.
    Format: bilangan_bulat + pembilang/penyebut
  • SHIFT + a b/c (d/c): Mengubah tampilan hasil pecahan antara bentuk improper dan campuran.
Tips dan Trik Penggunaan ▶
  • Implicit a: Kalkulator mendukung perkalian implisit.
    Contoh: 2π sama dengan 2×π
    Contoh: 5(3+2) sama dengan 5×(3+2)
  • Auto-scroll Display: Layar akan otomatis scroll ke kanan saat input panjang. Anda tetap bisa melihat bagian akhir yang sedang diketik.
  • Keyboard Shortcuts:
    • Angka 0-9: langsung ketik di keyboard
    • Operator: +, -, *, / di keyboard
    • Enter: sama dengan tombol =
    • Backspace: sama dengan DEL
    • Escape: sama dengan AC
  • Presisi Perhitungan: Kalkulator menggunakan presisi floating-point JavaScript (sekitar 15-17 digit signifikan).
  • Error Handling: Jika terjadi error (misalnya pembagian nol, domain error), layar akan menampilkan "Error". Tekan AC untuk reset.
Untuk Perhitungan Hisab Rukyat:
1. Gunakan mode DEG untuk sudut derajat
2. Manfaatkan format DMS untuk koordinat
3. Simpan hasil antara dengan Memory (STO/RCL)
4. Gunakan fungsi trigonometri untuk perhitungan posisi benda langit
